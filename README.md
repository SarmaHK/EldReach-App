# EldReach App

EldReach is a comprehensive IoT Dashboard and Room Architect designed to manage and monitor various devices across different environments seamlessly.

## Features
- **Room Architect**: Interactively design and structure room layouts with a grid-based builder.
- **IoT Dashboard**: Monitor live device status, network topologies, and simulated sensor data.
- **Settings**: Complete configuration module with options for user accounts and device connectivity.
- **Modern UI**: Fully responsive light/dark themes utilizing modern design aesthetics.

## Technologies
- React
- Vite
- Framer Motion

## Getting Started
First, install the dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.
